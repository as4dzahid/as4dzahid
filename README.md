🚧 Currently building

ResQAI 2.0 — an AI-powered disaster decision-support system. It turns scattered, real-time disaster data into fast, correct decisions:


Risk scoring — weighted multi-criteria model combining hazard proximity, disaster severity, and individual vulnerability
Shelter recommendation — multi-attribute ranking over distance, live occupancy, accessibility, and medical capability
Resource allocation — Hungarian-algorithm optimal assignment of volunteers to rescue requests under scarcity, benchmarked against a naive baseline
Evacuation routing — A* pathfinding over a hazard-weighted road graph


Built with FastAPI, SQLAlchemy, scipy, and a Python/JS decision-engine core with full test coverage.
